### Описание проекта  
Написать программу, которая угадывает загаданное компьютером число за минимальное количество попыток.  
Проект выполнен в рамках обучения на курсе [SkillFactory Data Science](https://skillfactory.ru/data-scientist).  

[:arrow_up: К оглавлению](#оглавление)

---

### Какой кейс решаем  
Нужно разработать алгоритм, который угадывает случайное число, загаданное компьютером.

**Условия задачи:**  
- Компьютер загадывает целое число от 1 до 100.  
- Программа должна угадать число как можно быстрее.  
- После каждой попытки программа получает подсказку: число больше или меньше загаданного.  

**Метрика качества:**  
Среднее количество попыток при 1000 повторений.

**Что практикуем:**  
- Разработка функций в Python.  
- Использование библиотек NumPy и случайных чисел.  
- Оптимизация алгоритмов поиска.
